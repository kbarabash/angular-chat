# npm команды

- help - выводит весь список команд
- lib-install - устанавливает все зависимости и конкатит их в libs.js
- register-files - регистрирует файлы в дев лоадер
- js-checker - проверяет валидность js файлов
- test - запускает юнит тесты
- codeCoverage - создает репорт по покрытию кода тестами
- build-css - билдит css в main.css
- watch - следит за изменениями и обновляет файлы
